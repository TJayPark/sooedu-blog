## 👋 학부모님께

오늘은 자녀분이 꼭 배워야 할 영어 표현, **set the table**를 소개합니다.

### 🎯 왜 이 표현이 중요한가요?
‘set the table’은 집에서 바로 써 볼 수 있는 생활 밀착형 표현이라 아이가 영어를 행동과 연결해 익히기 좋습니다. 단순히 단어를 외우는 것이 아니라 가족과 함께 식사 준비를 하며 자연스럽게 반복할 수 있어, 말하기 자신감과 생활 영어 감각을 함께 키워 줍니다.

# 📚 오늘의 표현: set the table

**발음:** `[셋 더 테이블]`  
**의미:** 식탁을 차리다

---

## 📖 Definition (영어 정의)
to put plates, cups, and utensils on the table before a meal

## 🗣️ 실전 대화 예시
> **Student:** Do you want me to set the table before dinner?  
> *(저녁 먹기 전에 제가 식탁 차릴까요?)*  

> **Teacher:** Yes, please. Set the table and put the spoons next to the bowls.  
> *(응, 부탁할게. 식탁을 차리고 숟가락은 그릇 옆에 놓아 줘.)*  


## 🔄 다양한 활용 (응용하기)
- **I set the table when my mom cooks dinner.**  
  *(엄마가 저녁을 하실 때 나는 식탁을 차려요.)*
- **Can you set the table for four people?**  
  *(4명 자리로 식탁 좀 차려 줄래?)*
- **Let’s set the table before our guests arrive.**  
  *(손님이 오시기 전에 식탁을 차리자.)*


## 💡 학부모 가이드
아이에게 식사 전마다 영어로 한 번씩 ‘Set the table, please.’라고 말해 보세요. 실제 행동과 함께 반복하면 표현이 훨씬 오래 기억됩니다. 가족 역할놀이로 ‘컵 놓기, 숟가락 놓기’까지 영어로 연결하면 효과가 더 좋습니다.

## 🎓 원어민 표현 팁
한국어식으로 ‘make the table’이라고 말하는 경우가 있는데, 자연스러운 표현은 ‘set the table’입니다. 또 ‘clean the table’은 식탁을 치우거나 닦는 뜻이라 의미가 다르므로 함께 구분해 주면 좋습니다.
